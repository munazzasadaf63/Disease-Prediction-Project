# 🧠 Disease Prediction System (Machine Learning)

This project is a machine learning-based system that predicts diseases based on user symptoms. It helps in early detection and provides useful insights.

---

## 📌 Features

* Predicts diseases using ML algorithms
* User-friendly interface
* Fast and accurate results
* Built using Python

---

## 🛠️ Technologies Used

* Python
* Machine Learning
* Libraries (NumPy, Pandas, Scikit-learn)
